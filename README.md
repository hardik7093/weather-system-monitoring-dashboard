# Real-Time Weather & System Monitoring Dashboard

## Overview
This project is a Linux-based monitoring system that collects weather data and system performance metrics and displays them on a web dashboard.

## Features
- Automated weather data collection using API (Bash & Python)
- CPU, RAM, Disk monitoring
- Cron jobs for automation
- MySQL database logging
- PHP dashboard with graphs

## Tech Stack
Linux | Bash | Python | Cron | PHP | MySQL | AWS EC2

## Workflow
API → Bash/Python Script → MySQL → PHP Dashboard

## Additional Scripts
- `weather_insert.sh` → Fetches weather data using API (Bash)
- `system.sh` → Collects system metrics (CPU, RAM, Disk)
- `weather.py` → Python-based weather data handling script
